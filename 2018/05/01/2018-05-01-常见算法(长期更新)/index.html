时间空间复杂度分析

对于每次迭代, 问题规模减少的数量
求数组的总和, 每次迭代,问题规模减少1, 一共需要迭代n次 -> O(n) 时间复杂度
排序数组中二分查找某个值, 每次迭代, 问题规模减少1/2 一共需要迭代 x次 x^2 = n, n = logN -> O(logN)
冒泡排序,外围需要n次迭代, 内部也需要n次迭代, 一共需要迭代n^2 -> O(n^2)

- 如何解决每次出队列的时候都要移动全部元素的问题?
- 定义两个指针分别指向下一个入队位置与出队位置,每次出队入队是将指针++,引发的问题: 容量不足
- 将两个指针对数组长度进行取模,使数组成为一个环形数组,这样指针正确就能容量正确
